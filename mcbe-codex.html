Player Tracker

Medium (4-6 lines)

Track nearby players with a compass that updates in real-time

/scoreboard objectives add tracker dummy /execute as @a run scoreboard players set @e[type=player,r=20] tracker 1 /execute as @a if score @s tracker matches 1 run tag @s add tracked /execute as @a[tag=!tracked] run scoreboard players set @s tracker 0 /execute as @a[tag=tracked] run tellraw @s {"rawtext":[{"text":"§aTracking: §e"}]} /tag @a remove tracked
Note: First command needs to run once. The rest should be in repeating command blocks in order.

Custom Crafting

Hard (7-9 lines)

Create custom crafting recipes with special output

/clear @a diamond 0 0 /execute as @a[nbt={Inventory:[{id:"minecraft:diamond",Count:1b,Slot:0b},{id:"minecraft:diamond",Count:1b,Slot:1b},{id:"minecraft:diamond",Count:1b,Slot:2b},{id:"minecraft:diamond",Count:1b,Slot:3b},{id:"minecraft:diamond",Count:1b,Slot:4b},{id:"minecraft:diamond",Count:1b,Slot:5b},{id:"minecraft:diamond",Count:1b,Slot:6b},{id:"minecraft:diamond",Count:1b,Slot:7b},{id:"minecraft:diamond",Count:1b,Slot:8b}]}] run give @s minecraft:netherite_ingot 1 /execute as @a[nbt={Inventory:[{id:"minecraft:diamond",Count:1b,Slot:0b},{id:"minecraft:diamond",Count:1b,Slot:1b},{id:"minecraft:diamond",Count:1b,Slot:2b},{id:"minecraft:diamond",Count:1b,Slot:3b},{id:"minecraft:diamond",Count:1b,Slot:4b},{id:"minecraft:diamond",Count:1b,Slot:5b},{id:"minecraft:diamond",Count:1b,Slot:6b},{id:"minecraft:diamond",Count:1b,Slot:7b},{id:"minecraft:diamond",Count:1b,Slot:8b}]}] run clear @s diamond 0 9
This creates a recipe where 9 diamonds in a crafting grid gives 1 netherite ingot. Place in repeating command block.

Random Teleport

Easy (1-3 lines)

Teleport to a random location within specified bounds

Teleports a random player within 100-200 blocks of the command block. Change values as needed.

Item Sorter

Medium (4-6 lines)

Submitted by: CommandMaster42

/execute as @e[type=item] at @s run tag @s add toSort /execute as @e[tag=toSort,nbt={Item:{id:"minecraft:dirt"}}] at @s run tp @s ~~~10 /execute as @e[tag=toSort,nbt={Item:{id:"minecraft:stone"}}] at @s run tp @s ~~~-10 /tag @e[type=item] remove toSort
Sorts dirt to +10 Z and stone to -10 Z from dropper. Add more items as needed.
